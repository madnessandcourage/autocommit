# Autocommit

A ChatGPT-powered commit messages, branch names et cetera

## How to install

```sh
pipx install git+https://github.com/madnessandcourage/autocommit.git
```

## How to updade

```sh
pipx upgrade autocommit
```

## How to use

1. Do some changes
2. `git add` them
3. `git co` to checkout to new branch
4. `git ci` to commit
